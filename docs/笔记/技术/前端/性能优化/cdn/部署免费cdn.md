## 1. 选择方案
使用github和jsdelivr这对组合, 搭建免费的cdn, 不仅节省存储费用, 还节省带宽费用

::: tip
使用github的存储, 使用jsdelivr的cdn带宽
:::

## 2. 部署步骤
### 2.1 github上创建仓库
注意是public类型的

### 2.2 使用jsdelivr的cdn地址访问
将资源的github访问url转换为jsdelivr的cdn url
