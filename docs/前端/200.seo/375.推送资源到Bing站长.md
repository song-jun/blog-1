# seo 优化-推送资源到 Bing 站长

操作过程与其它站长雷同，此处不再赘述。  
Bing 站长不支持自动推送。  
与其它不同站长不同的是，Bing 站长可以直接从 google 导入站点的相关数据，这样可免去站点的验证和资源提交操作。
