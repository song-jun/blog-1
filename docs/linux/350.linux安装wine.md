# linux 安装 wine

## 一. 准备环境

CentOS 6.5 x86_64

## 二. 安装

wget http://download.fedoraproject.org/pub/epel/6/x86_64/epel-release-6-8.noarch.rpm  
rpm -ivh ./epel-release-6-8.noarch.rpm  
yum install wine  
yum groupinstall desktop
