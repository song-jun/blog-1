# mysql 类型转换函数

mysql 提供了 2 个类型转换函数，功能相同，仅语法不同。

## 使用 CAST 函数进行类型转换

- 语法  
  CAST(xxx AS 类型)

- 示例

```
select cast("1" as signed);
```

## 使用 CONVERT 函数进行类型转换

- 语法  
  CONVERT(xxx,类型)
- 示例

```
select convert("1", signed );
```
