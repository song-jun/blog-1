# springboot 接口参数校验

有 2 种方式:

### 单个接口参数上增加校验

- 在 controller 类上添加注解@Validated
- 在 api 参数前添加参数校验注解

### 实体类上增加校验

- 在实体类对应的 controller 参数前添加注解@Validated
- 在实体类的属性上添加校验注解
