# vuepress 支持搜索

默认主题自带的搜索, 只会为页面的**标题、h2、h3 以及 tags**构建搜索索引。所以尽量将围绕知识点的关键字体现到标题上。而 tags 更为灵活，可以把相关的能想到的关键字都配置到 tags 中，以方便搜索。

如果期望全文搜索，请前往[vuepress 支持全文搜索](/vuepress-fulltext-search.html)
